Onze modal

In JavaScript kan je een dialog openen en sluiten met de open attribuut. We selecteren hiervoor de gepaste modal, omdat we efficient programmeren hardcoderen we de ID niet in JavaScript maar gebruiken we een data attribuut op de button.

Maak hier onder een nieuwe modal en knop om deze te openen. Plaats een attribuut data-modal op de button die de modal opent.
