Dialog en Popover API

Popover is een element dat gebruikt wordt om overlays te maken, het woord 'popover' is een soort van koepelterm. Dialog is namelijk een soort van popover die specifiek bedoeld is voor modals (grotere vensters). Bij popover is het belangrijk dat je de correcte aria-attributes gebruikt.

Meer over Dialogs en Popovers

Verschillen?

Een dialog trapt de focus en wordt voornamelijk gebruikt voor user-input (formulieren, bevestigen van een actie, verwijderen, ...). Popover API blokkeert geen achterliggende inhoud en wordt eerder gebruikt voor kleinere, contextuele informatie. Ook kan het popover-attribuut gebruikt worden op ieder element en is er meer controle via JavaScript mogelijk.
