Random properties

Dit zijn CSS properties en values die je kunt gebruiken om je website responsief te maken zonder media queries. Sommige van deze properties zijn moderner en zorgen voor 'kortere' code.

min()

De min() functie kiest de kleinste waarde uit de opgegeven lijst en vergelijkt dit met de breedte van de container/parent.

inline en block

In plaats van margin-left en margin-right of padding-left en padding-right kun je ook gebruik maken van inline-schrijfwijze.

In plaats van margin-top en margin-bottom of padding-top en padding-bottom kun je ook gebruik maken van block-schrijfwijze.

Dit kan handig zijn om sneller en korter code te schrijven.
