View Transitions

Animeren tussen pagina's? Hiervoor moeten we eerst het verschil kennen tussen MPA's en SPA's.

Wat zijn MPA's en SPA's?

Een SPA is een Single Page Application waarbij de pagina niet volledig of "echt" wordt herladen bij navigatie, maar alleen de benodigde content wordt bijgewerkt, dit via JavaScript.

Een MPA is een Multi Page Application waarbij elke pagina een aparte HTML-pagina is die volledig wordt herladen bij navigatie. Dit is een zogenaamde 'hard reload'.

Animatie tussen pagina's

Dankzij de nieuwe View Transitions API kunnen we nu animaties maken tussen pagina's zonder dat we een SPA hoeven te gebruiken.
