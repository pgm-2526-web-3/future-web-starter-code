Je kan simpele view transition toevoegen tussen pagina's dankzij de View Transitions API. Standaard zal er een fade effect zijn, maar je kan dit aanpassen met CSS en jouw eigen animaties op de pseudo-elementen.

In de inspector kan je de view transition pseudo-elementen bekijken. (Open de animations tab om de animaties te zien. Cmd/Ctrl + SHIFT + P)

Merk op dat standaard de volledige pagina wordt geanimeerd (root). Je kan dit aanpassen door specifieke elementen te markeren met view-transition-name in CSS of via JavaScript.
